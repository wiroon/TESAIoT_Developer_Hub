# สลับ LED (LED Toggle)

ตัวอย่างนี้สร้างปุ่ม toggle ตัวเดียวที่สลับสถานะ LED ฮาร์ดแวร์ทุกครั้งที่กด Virtual LED บนหน้าจอซิงโครไนซ์กับสถานะ LED จริง

ฟังก์ชัน `cyhal_gpio_toggle` กลับสถานะขา GPIO ปัจจุบันโดยไม่ต้องติดตามค่าปัจจุบัน ตัวแปรสถานะซอฟต์แวร์ติดตามสถานะตรรกะเพื่อให้ UI label แม่นยำ

วิธีนี้ง่ายกว่ารูปแบบ ON/OFF เนื่องจากต้องการปุ่มเพียงตัวเดียว label ปุ่มอัปเดตระหว่าง "ON" และ "OFF" เพื่อสะท้อนสถานะปัจจุบัน สี Virtual LED เปลี่ยนจากเขียว (เปิด) เป็นแดง (ปิด)

รูปแบบ toggle ใช้กันทั่วไปในอุปกรณ์ IoT สำหรับสวิตช์เปิดปิด ควบคุมรีเลย์ และเลือกโหมด รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
