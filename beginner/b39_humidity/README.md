# เกจความชื้น (Humidity Gauge)

ตัวอย่างนี้อ่านความชื้นสัมพัทธ์จากเซ็นเซอร์สภาพอากาศ SHT40 และแสดงใน Arc gauge ช่วงเกจครอบคลุม 0 ถึง 100 เปอร์เซ็นต์ความชื้นสัมพัทธ์ พร้อม label ค่าตรงกลางขนาดใหญ่

SHT40 ให้ทั้งค่าอุณหภูมิและความชื้นผ่าน `sht40_read` ตัวอย่างนี้เน้นค่าความชื้น ค่าถูกอ่านทุก 1000ms เนื่องจากความชื้นเปลี่ยนแปลงอย่างค่อยเป็นค่อยไป

ตัวบ่งชี้ Arc เปลี่ยนสีตามระดับความชื้น: น้ำเงินสำหรับสภาพแห้ง (ต่ำกว่า 30%) เขียวสำหรับสบาย (30-60%) และแดงสำหรับชื้น (สูงกว่า 60%) ข้อความตัวบ่งชี้ความสบายด้านล่างให้การประเมินที่อ่านเข้าใจง่าย

ตัวอย่างนี้รองรับ TESAIoT Dev Kit และ PSoC Edge AI Kit เท่านั้น (`#if BSP_HAS_SHT40`) เนื่องจาก Eva Kit ไม่มีเซ็นเซอร์ความชื้น SHT40
