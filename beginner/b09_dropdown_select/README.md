# เมนูเลือก Dropdown (Dropdown Select)

ตัวอย่างนี้สร้างเมนู Dropdown ที่มีรายชื่อเซ็นเซอร์ เมื่อผู้ใช้เลือกรายการจาก Dropdown Label ด้านล่างจะอัปเดตแสดงเซ็นเซอร์ที่เลือก

ตัวเลือก Dropdown ถูกกำหนดเป็นข้อความคั่นด้วยบรรทัดใหม่ผ่าน `lv_dropdown_set_options` เซ็นเซอร์ที่มีให้เลือก ได้แก่ BMI270, DPS368, SHT40, BMM350 และ Radar callback `LV_EVENT_VALUE_CHANGED` อ่านข้อความที่เลือกด้วย `lv_dropdown_get_selected_str`

Dropdown widget เหมาะสำหรับหน้าจอตั้งค่าที่ผู้ใช้ต้องเลือกจากรายการที่กำหนดไว้ ใช้พื้นที่หน้าจอน้อยเมื่อปิดอยู่ และขยายแสดงตัวเลือกทั้งหมดเมื่อแตะ

ตัวอย่างนี้รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit เนื่องจากแสดงเฉพาะตัวเลือกข้อความโดยไม่ได้อ่านฮาร์ดแวร์เซ็นเซอร์จริง
