# ค่า Gyroscope XYZ (Gyroscope XYZ)

ตัวอย่างนี้อ่านข้อมูล gyroscope จากเซ็นเซอร์ IMU BMI270 แสดงความเร็วเชิงมุมของแกน X, Y, Z ในหน่วยองศาต่อวินาที timer 100ms ให้การอัปเดต real-time ที่ลื่นไหล

Gyroscope วัดความเร็วในการหมุนรอบแต่ละแกน ขณะอยู่นิ่งค่าทั้งหมดควรใกล้ศูนย์ การหมุนบอร์ดจะให้ค่าที่สัดส่วนกันบนแกนที่เกี่ยวข้อง BMI270 ให้ช่วง gyro สูงสุด 2000 องศาต่อวินาที

Arc gauge สามตัวให้การแสดงผลที่เข้าใจง่ายของอัตราการหมุน พร้อม label ตรงกลางแสดงค่าตัวเลข Arc ใช้ช่วงสมมาตรที่ศูนย์กลาง การหมุนบวกแสดงในทิศทางหนึ่งและลบในอีกทิศทาง

BMI270 gyroscope มีทั้งบน AI Kit และ Eva Kit รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
