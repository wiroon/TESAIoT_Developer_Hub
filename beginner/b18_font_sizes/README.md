# ขนาดฟอนต์ (Font Sizes)

ตัวอย่างนี้แสดงข้อความเดียวกันในขนาดฟอนต์ Montserrat ห้าขนาดที่ใช้ได้ทั้งบน AI Kit และ Eva Kit: 14, 16, 20, 24 และ 28 pixel เป็นข้อมูลอ้างอิงภาพสำหรับเลือกขนาดฟอนต์ที่เหมาะสมกับ UI ของคุณ

แต่ละ label จัดเรียงแนวตั้งใน flex column พร้อมแสดงชื่อฟอนต์และขนาดคู่กับข้อความตัวอย่าง label ใช้ `lv_obj_set_style_text_font` เพื่อเลือกรุ่น Montserrat ในตัวที่เหมาะสม

ขนาดฟอนต์ที่ใช้ได้ถูกกำหนดจากสิ่งที่ compile ใน LVGL configuration ทั้ง AI Kit และ Eva Kit มี Montserrat 14, 16, 20, 24 และ 28 ขนาดเพิ่มเติมอาจมีในบอร์ดเฉพาะ

การเลือกขนาดฟอนต์ที่เหมาะสมสำคัญต่อความสามารถในการอ่านบนจอแสดงผล PSoC Edge ขนาด 14-16 เหมาะสำหรับ label, 20 สำหรับเนื้อหา และ 24-28 สำหรับหัวข้อ รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
