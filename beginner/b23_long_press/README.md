# การตรวจจับกดค้าง (Long Press Detection)

ตัวอย่างนี้สาธิตการแยกแยะระหว่างการโต้ตอบแบบสัมผัสสามประเภท: กดสั้น กดค้าง (ค้างนานกว่า 400ms) และกดค้างซ้ำ (ส่งต่อเนื่องขณะค้าง) แต่ละประเภท event trigger การตอบสนองทางภาพที่แตกต่างกัน

ปุ่มตัวเดียวลงทะเบียน `LV_EVENT_ALL` และใช้ switch-case ใน callback เพื่อระบุ event code เฉพาะ click event เพิ่มตัวนับ กดค้างเปลี่ยนสีปุ่ม และกดค้างซ้ำเพิ่มตัวนับ rapid-fire แยก

output panel แสดงประเภท event ล่าสุดและตัวนับที่เกี่ยวข้อง ให้ feedback ชัดเจนว่าตรวจจับ gesture ใด สิ่งนี้จำเป็นสำหรับสร้าง interface ที่มี shortcut สำหรับผู้ใช้ขั้นสูง เช่น กดค้างเพื่อลบ หรือค้างเพื่อเลื่อน

เวลากดค้างกำหนดได้ผ่านการตั้งค่า input device ของ LVGL รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
