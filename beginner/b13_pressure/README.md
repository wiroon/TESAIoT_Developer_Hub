# แถบความดัน (Pressure Bar)

ตัวอย่างนี้อ่านความดันบรรยากาศจากเซ็นเซอร์ DPS368 และแสดงด้วย Bar widget ช่วง bar ครอบคลุม 950 ถึง 1050 hPa ครอบคลุมการเปลี่ยนแปลงความดันบรรยากาศที่ระดับน้ำทะเลทั่วไป

DPS368 ให้ค่าความดันที่มีความแม่นยำสูง เหมาะสำหรับ monitor สภาพอากาศ ประมาณความสูง และนำทางในอาคาร ความดันถูกอ่านพร้อมอุณหภูมิผ่าน `dps368_read_both` เนื่องจากเซ็นเซอร์ต้องใช้การชดเชยอุณหภูมิ

label ตัวเลขขนาดใหญ่แสดงค่าความดันที่แม่นยำในหน่วย hectopascal (hPa) bar ให้ตัวบ่งชี้ภาพว่าค่าที่อ่านได้อยู่ตรงไหนในช่วงที่คาดหวัง ความดันบรรยากาศมาตรฐานที่ระดับน้ำทะเลอยู่ที่ประมาณ 1013.25 hPa

ตัวอย่างนี้รองรับ TESAIoT Dev Kit และ PSoC Edge AI Kit เท่านั้น (`#if BSP_HAS_DPS368`) เนื่องจาก Eva Kit ไม่มีเซ็นเซอร์ DPS368
