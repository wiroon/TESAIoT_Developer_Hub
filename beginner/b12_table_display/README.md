# แสดงตาราง (Table Display)

ตัวอย่างนี้สร้างตารางข้อมูลแสดงค่าเซ็นเซอร์ในรูปแบบ grid ที่มีโครงสร้าง ตารางมี 4 แถวและ 3 คอลัมน์: ชื่อเซ็นเซอร์ ค่า และหน่วย แถวแรกทำหน้าที่เป็นส่วนหัวด้วยสีพื้นหลังที่แตกต่าง

ค่าของเซลล์ถูกกำหนดด้วย `lv_table_set_cell_value` พร้อมดัชนีแถวและคอลัมน์ แถวส่วนหัวจัดรูปแบบอย่างเด่นชัดโดยใช้ style property ระดับเซลล์เพื่อสร้างการแยกทางภาพจากแถวข้อมูล

ความกว้างคอลัมน์ถูกกำหนดแยกเพื่อรองรับเนื้อหาที่มีความยาวต่างกัน คอลัมน์เซ็นเซอร์กว้างขึ้นเพื่อใส่ชื่อ ในขณะที่คอลัมน์หน่วยแคบ สาธิตการจัดวาง table ที่ใช้งานจริงสำหรับ IoT dashboard ที่ต้องแสดงค่าเซ็นเซอร์หลายตัวพร้อมกัน

ตารางแสดงค่าตัวอย่างที่จะถูกแทนที่ด้วยค่าเซ็นเซอร์จริงในแอปพลิเคชันจริง รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
