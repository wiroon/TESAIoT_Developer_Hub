# B01 — Widget Basics (Grid Menu)

Explore 11 fundamental LVGL widgets through an interactive grid menu.
Tap any card to see a live demo, then tap "Home" to return.

## Widgets Covered
- **Hello World** — Label creation and styling
- **Button** — Click events and counter
- **Arc Gauge** — Circular gauge with animation
- **Checkbox** — Multiple option selection
- **Switch** — On/Off toggle
- **Progress Bar** — Loading animation
- **Spinner** — Indeterminate loading
- **Dropdown** — Selection menu
- **Text Input** — Textarea with keyboard
- **Slider** — Value range selection
- **Table** — Data grid display

## Files
- `main_example.c` — Grid menu framework + navigation
- `pages.h` — Sub-page declarations
- `page_*.c` — Individual widget demos (11 files)

## How to Use
1. Copy all files to your MTB template's `proj_cm55/app/` directory
2. Build with `make build -j`
3. The grid menu appears on screen — tap any card to explore
