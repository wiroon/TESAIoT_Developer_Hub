# B02 — Widget Applied (Grid Menu)

10 practical examples showing widgets connected to real functionality:
sensors, GPIO, timers, and interactive games.

## Demos
- **LED Brightness** — Slider controls LED brightness (arc gauge)
- **Switch LED** — Toggle switch with LED widget
- **Stopwatch** — Start/Stop/Reset timer
- **Reaction Game** — Test your reaction time
- **Bar Chart** — Random data visualization
- **IMU Display** — BMI270 accelerometer live data
- **Temperature** — DPS368/SHT40 reading with gauge
- **Humidity** — SHT40 humidity bar
- **Compass** — BMM350 heading with cardinal direction
- **System Info** — Uptime, heap, task count

## Board Compatibility
- AI Kit (KIT_PSE84_AI) — all demos
- Eva Kit (KIT_PSE84_EVAL_EPC2) — IMU, Compass, System Info; no DPS368/SHT40
