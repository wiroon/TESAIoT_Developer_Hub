# อ่านปุ่มฮาร์ดแวร์ (Hardware Button Read)

ตัวอย่างนี้อ่านปุ่มกด SW1 และ SW2 ที่ต่อกับขา GPIO P5_2 และ P5_3 LVGL timer poll สถานะปุ่มทุก 50ms และอัปเดตหน้าจอแสดงสถานะ PRESSED หรือ Released

ขา GPIO ถูกเริ่มต้นเป็น input พร้อมตัวต้านทาน pull-up ภายในด้วย `CYHAL_GPIO_DRIVE_PULLUP` ปุ่มเป็นแบบ active-low หมายความว่า `cyhal_gpio_read` คืนค่า 0 เมื่อกดและ 1 เมื่อปล่อย

label สถานะสองตัวแสดงสถานะปัจจุบันของแต่ละปุ่มด้วยข้อความที่มีรหัสสี: เขียวสำหรับกด และเทาสำหรับปล่อย ตัวนับการกดติดตามจำนวนการกดทั้งหมดที่ตรวจจับได้สำหรับแต่ละปุ่ม

การอ่าน input แบบ polling ง่ายกว่าแบบ interrupt และเพียงพอสำหรับแอปพลิเคชัน UI ที่ latency 50ms เป็นที่ยอมรับได้ รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
