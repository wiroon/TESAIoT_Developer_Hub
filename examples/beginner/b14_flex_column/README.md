# เลย์เอาต์แนวตั้ง (Flex Column Layout)

ตัวอย่างนี้จัดเรียงรายการที่มี label ห้ารายการแนวตั้งโดยใช้ `LV_FLEX_FLOW_COLUMN` แต่ละรายการใช้ `lv_obj_set_flex_grow` เพื่อแบ่งพื้นที่แนวตั้งตามสัดส่วน

flex grow factor กำหนดวิธีกระจายพื้นที่ที่เหลือระหว่างรายการ รายการที่มีค่า grow สูงกว่าจะได้รับพื้นที่มากกว่าตามสัดส่วน ในตัวอย่างนี้ พื้นที่เนื้อหาหลักมีค่า grow ใหญ่กว่าส่วนหัวและส่วนท้าย

row padding ถูกกำหนดด้วย `lv_obj_set_style_pad_row` เพื่อเพิ่มช่องว่างแนวตั้งระหว่างรายการ แต่ละแถวมี label พร้อมแถบพื้นหลังสี สร้างลักษณะคล้ายเมนู

flex column เหมาะอย่างยิ่งสำหรับสร้างรายการที่เลื่อนแนวตั้ง เมนูตั้งค่า และ panel dashboard แบบซ้อน รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
