# เกจ Arc (Arc Gauge)

ตัวอย่างนี้สร้าง Arc gauge widget ที่แสดงค่า 0 ถึง 100 โดย Label ตรงกลาง Arc แสดงค่าตัวเลขปัจจุบัน LVGL timer เพิ่มค่าอัตโนมัติทุก 50ms สร้างเอฟเฟกต์ animation ที่ลื่นไหล

มุมพื้นหลังของ Arc กำหนดด้วย `lv_arc_set_bg_angles` เพื่อสร้างการกวาด 270 องศาจาก 135 ถึง 45 องศา ช่วงค่าตั้งจาก 0 ถึง 100 ด้วย `lv_arc_set_range` การหมุนถูกตั้งให้ Arc เริ่มจากด้านซ้ายล่าง

เมื่อค่าถึง 100 timer จะรีเซ็ตกลับเป็น 0 สร้างการวนซ้ำอย่างต่อเนื่อง Label ตรงกลางอัปเดตทุกรอบเพื่อแสดงค่าเปอร์เซ็นต์ รูปแบบนี้ใช้กันทั่วไปใน dashboard เซ็นเซอร์ที่แสดงอุณหภูมิ ความชื้น หรือตัวบ่งชี้ความคืบหน้า
