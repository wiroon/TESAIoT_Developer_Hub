# ตัวหมุนรอโหลด (Spinner Loading)

ตัวอย่างนี้สาธิต Spinner widget ของ LVGL ซึ่งให้ตัวบ่งชี้การโหลดแบบ animation วงกลม Spinner ถูกสร้างด้วยพารามิเตอร์ระยะเวลาการหมุนและความยาว arc ที่กำหนดได้

Spinner ถูกจัดรูปแบบด้วยสี arc ที่กำหนดเองสำหรับส่วนตัวบ่งชี้ที่ active ในขณะที่ track พื้นหลังใช้สีอ่อนกว่า ขนาดถูกกำหนดอย่างชัดเจนเพื่อสร้างตัวบ่งชี้การโหลดที่เด่นชัดตรงกลางหน้าจอ

Label ด้านล่าง Spinner แสดง "Loading..." เพื่อให้บริบทข้อความควบคู่กับ animation ในแอปพลิเคชันจริง Label นี้จะอัปเดตเพื่อแสดงรายละเอียดความคืบหน้า เช่น "กำลังเชื่อมต่อ WiFi..." หรือ "กำลังปรับเทียบเซ็นเซอร์..."

ตัวอย่างนี้เป็นการแสดงผลอย่างเดียว รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
