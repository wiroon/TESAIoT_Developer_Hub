# B02 — Color Cards

Four colored cards arranged in a 2x2 grid on the display, each with a centered text label.

## What it demonstrates

- Creating styled card containers with `example_card_create()`
- Flex layout with `LV_FLEX_FLOW_ROW_WRAP` for automatic grid arrangement
- Using custom colors for card backgrounds
- Padding and gap configuration

## Hardware

- Board: KIT_PSE84_AI / KIT_PSE84_EVAL_EPC2
- Sensors: None

## How to use

1. Copy `main_example.c` to `proj_cm55/app/` replacing the existing file
2. Build and flash: `make build -j && make program`
