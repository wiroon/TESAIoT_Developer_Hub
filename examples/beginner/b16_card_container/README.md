# การ์ดคอนเทนเนอร์ (Card Container)

ตัวอย่างนี้สร้าง container สไตล์การ์ดสามใบตามหลัก Material Design แต่ละการ์ดมีมุมโค้งมน เงาที่ละเอียดอ่อน และขอบบางเพื่อสร้างความลึกและการแยกทางภาพ

การ์ดจัดเรียงในแถว flex แนวนอน แต่ละการ์ดมี label หัวข้อ การแสดงค่า และคำบรรยายย่อย เอฟเฟกต์เงาใช้ `lv_obj_set_style_shadow_width` และ `lv_obj_set_style_shadow_opa` เพื่อลักษณะยกขึ้นที่สมจริง

border radius ตั้งเป็น 12 pixel ด้วย `lv_obj_set_style_radius` สำหรับมุมโค้งมนที่นุ่มนวล แต่ละการ์ดมีสีเน้นแตกต่างที่ขอบด้านบนเพื่อแยกหมวดหมู่ข้อมูล

รูปแบบการ์ดนี้เป็นรากฐานสำหรับ IoT dashboard ที่แสดงข้อมูลเซ็นเซอร์ในรูปแบบที่เป็นระเบียบและน่าสนใจ รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
