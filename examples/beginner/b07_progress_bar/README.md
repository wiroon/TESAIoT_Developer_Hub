# แถบความคืบหน้า (Progress Bar)

ตัวอย่างนี้แสดง Progress bar แนวนอนที่เติมจาก 0 ถึง 100 เปอร์เซ็นต์ timer callback เพิ่มค่า bar ทีละ 1 ทุก 100ms และ Label ด้านบน bar แสดงเปอร์เซ็นต์ปัจจุบัน

Bar widget ใช้ `lv_bar_set_range` เพื่อกำหนดช่วง 0-100 และ `lv_bar_set_value` ร่วมกับ `LV_ANIM_ON` สำหรับการเปลี่ยนแปลงแบบ animation ที่ลื่นไหล เมื่อ bar ถึง 100% จะรีเซ็ตเป็น 0 โดยอัตโนมัติเพื่อเริ่มรอบใหม่

Bar ถูกจัดรูปแบบด้วยสี indicator แบบกำหนดเองจากระบบ Palette ของ LVGL ความกว้างครอบคลุมเกือบเต็มหน้าจอเพื่อให้ตัวบ่งชี้ภาพชัดเจน รูปแบบนี้ใช้กันทั่วไปสำหรับติดตามความคืบหน้าการดาวน์โหลด การปรับเทียบเซ็นเซอร์ หรือการทำงานเสร็จสมบูรณ์

ตัวอย่างนี้รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
