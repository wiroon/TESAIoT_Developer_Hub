# เกจอุณหภูมิ (Temperature Gauge)

ตัวอย่างนี้อ่านอุณหภูมิจากเซ็นเซอร์ความดันบรรยากาศ DPS368 และแสดงใน Arc gauge ช่วงเกจครอบคลุม 0 ถึง 50 องศาเซลเซียส เหมาะสำหรับสภาพแวดล้อมในอาคาร label ตรงกลางขนาดใหญ่แสดงค่าตัวเลขที่แม่นยำ

เซ็นเซอร์ DPS368 ให้ทั้งค่าอุณหภูมิและความดันผ่าน `dps368_read_both` ตัวอย่างนี้ใช้เฉพาะค่าอุณหภูมิ ทิ้งค่าความดัน เซ็นเซอร์ถูกอ่านทุก 500ms ซึ่งเพียงพอสำหรับ monitor อุณหภูมิ เนื่องจากอุณหภูมิเปลี่ยนแปลงช้า

สี Arc gauge ไล่จากน้ำเงิน (เย็น) ผ่านเขียว (สบาย) ไปแดง (ร้อน) ตามอุณหภูมิที่เพิ่มขึ้น ให้การแสดงภาพที่เข้าใจง่ายของสถานะความร้อน

ตัวอย่างนี้รองรับ TESAIoT Dev Kit และ PSoC Edge AI Kit เท่านั้น (`#if BSP_HAS_DPS368`) เนื่องจาก Eva Kit ไม่มีเซ็นเซอร์ DPS368
