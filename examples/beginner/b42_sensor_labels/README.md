# Dashboard เซ็นเซอร์ทั้งหมด (All Sensors Dashboard)

ตัวอย่างนี้สร้าง dashboard เซ็นเซอร์ครบถ้วนที่ปรับตัวอัตโนมัติตามฮาร์ดแวร์ที่มีโดยใช้ BSP feature guard บน AI Kit จะแสดงค่า BMI270, DPS368 และ SHT40 บน Eva Kit จะแสดงค่า BMI270, CapSense และ Potentiometer

แต่ละส่วนเซ็นเซอร์ถูกครอบด้วย `#if BSP_HAS_*` preprocessor guard รับรองว่าเฉพาะเซ็นเซอร์ที่มีจะถูกเริ่มต้นและแสดงผล เลย์เอาต์ปรับตัวแบบ dynamic ตามจำนวนส่วนเซ็นเซอร์ที่ active

dashboard ใช้เลย์เอาต์แบบการ์ด แต่ละเซ็นเซอร์มี panel ของตัวเอง การ์ดจัดเรียงใน flex row ที่ wrap ไปบรรทัดถัดไปเมื่อเกินความกว้างหน้าจอ แต่ละการ์ดแสดงชื่อเซ็นเซอร์ ค่าปัจจุบัน และตัวบ่งชี้สถานะเล็กๆ

ตัวอย่างนี้สาธิตวิธีสร้าง firmware ที่เข้ากันได้ข้ามบอร์ดที่จัดการความแตกต่างของฮาร์ดแวร์อย่างสง่างาม compile และรันได้อย่างถูกต้องบนทั้ง AI Kit และ Eva Kit รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
