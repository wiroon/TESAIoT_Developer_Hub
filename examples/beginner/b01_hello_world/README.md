# ป้ายข้อความ Hello World (Hello World Label)

ตัวอย่างนี้สาธิตการทำงานพื้นฐานที่สุดของ LVGL: การสร้างและจัดรูปแบบ Label สามตัว โดยวางไว้บนหน้าจอในตำแหน่งที่ต่างกัน ใช้ `lv_obj_align` เพื่อแสดงการจัดตำแหน่งแบบบน กลาง และล่าง

แต่ละ Label ใช้สีที่แตกต่างจากระบบ Palette ของ LVGL โดยหัวข้อใช้สีน้ำเงิน ข้อความหลักใช้สีเขียว และคำอธิบายใช้สีเทา สีถูกกำหนดผ่าน `lv_obj_set_style_text_color` ร่วมกับ `lv_palette_main()` เพื่อให้ธีมสีสะอาดและสม่ำเสมอ

ข้อความทักทาย "Hello BENTO!" แสดงด้วยฟอนต์ขนาดใหญ่ตรงกลางหน้าจอ โดยกำหนด text font style property เป็น `&lv_font_montserrat_28` การเปลี่ยนขนาดฟอนต์ใน LVGL เป็นการตั้งค่า style property ไม่ใช่การเรียก API แยก

ตัวอย่างนี้รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit เนื่องจากใช้เพียงการวาด LVGL พื้นฐาน ไม่มีการพึ่งพาเซ็นเซอร์หรืออุปกรณ์ต่อพ่วง เหมาะเป็นจุดเริ่มต้นสำหรับตัวอย่างทั้งหมดในชุดนี้
