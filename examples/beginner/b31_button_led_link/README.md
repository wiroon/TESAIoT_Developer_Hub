# เชื่อมปุ่มกับ LED (Button-LED Link)

ตัวอย่างนี้เชื่อมปุ่มฮาร์ดแวร์กับ LED ฮาร์ดแวร์โดยตรง เมื่อกดปุ่ม SW1 บนขา P5_2 LED บนขา P13_7 จะเปิด เมื่อปล่อย LED จะปิด UI สะท้อนสถานะ real-time ด้วย Virtual LED และ label สถานะ

timer 50ms poll สถานะปุ่มและเขียนผลลัพธ์ไปยัง LED GPIO โดยตรง สร้างการเชื่อมระดับฮาร์ดแวร์ที่ LED ตามปุ่มด้วย latency ต่ำสุด หน้าจอ LVGL อัปเดตพร้อมกัน

นี่คือรูปแบบการเชื่อม input-output GPIO ที่ง่ายที่สุด สาธิตรูปแบบพื้นฐานที่ใช้ในแอปพลิเคชันควบคุม embedded ทั้งหมด ปุ่มให้สัญญาณ input ตรรกะประมวลผลกำหนดการตอบสนอง และ LED ให้ output

รูปแบบเดียวกันขยายไปยังสถานการณ์ที่ซับซ้อนกว่า เช่น ควบคุมรีเลย์ เปิดมอเตอร์ และ trigger สัญญาณเตือน รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
