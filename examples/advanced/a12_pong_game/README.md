# เกมปิงปอง (Pong Game)

ตัวอย่างนี้สร้างเกม Pong แบบคลาสสิกพร้อมฟิสิกส์ลูกบอล โหมดผู้เล่นปะทะ AI และการนับคะแนนถึง 11 แต้ม ผู้เล่นควบคุมไม้ตีด้านซ้ายโดยใช้จอยสติ๊ก F310 หรือ D-pad แบบสัมผัส ขณะที่ AI ควบคุมไม้ตีด้านขวาพร้อมระดับความยากที่ปรับได้

ฟิสิกส์ลูกบอลใช้ตำแหน่งและเวกเตอร์ความเร็วแบบทศนิยมเพื่อการเคลื่อนที่ที่ลื่นไหล ลูกบอลสะท้อนจากกำแพงด้านบนและด้านล่างโดยรักษามุม และสะท้อนจากไม้ตีด้วยมุมที่ขึ้นอยู่กับตำแหน่งที่กระทบหน้าไม้ตี การตีตรงกลางจะให้การสะท้อนในแนวนอน ขณะที่การตีที่ขอบจะให้มุมชันขึ้น ทำให้ผู้เล่นควบคุมทิศทางได้

AI ฝ่ายตรงข้ามติดตามตำแหน่ง Y ของลูกบอลพร้อมการหน่วงเวลาและจำกัดความเร็วสูงสุดเพื่อสร้างระดับความยากที่เอาชนะได้ การทำนายของ AI ใช้วิถีลูกบอลเพื่อเคลื่อนไม้ตีล่วงหน้า แต่มีข้อผิดพลาดสุ่มเล็กน้อยตามสัดส่วนของค่าความยากเพื่อป้องกันการเล่นที่สมบูรณ์แบบ

การเรนเดอร์ใช้ LVGL Canvas พร้อม Primitive สี่เหลี่ยมสำหรับไม้ตีและวงกลมสำหรับลูกบอล คะแนนแสดงเป็นข้อความขนาดใหญ่ตรงกลางด้านบน เมื่อผู้เล่นฝ่ายใดถึง 11 แต้ม จะแสดงหน้าจอชนะ/แพ้พร้อมปุ่มเริ่มใหม่ เส้นประตรงกลางแบ่งสนาม
