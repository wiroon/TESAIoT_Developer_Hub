# ตัวแสดงผล Joystick (Joystick Visualizer)

ตัวอย่างนี้แสดงผลภาพอินพุต joystick แบบ dual-analog อย่างสมบูรณ์ พื้นที่วงกลม 2 วงแทนสติ๊กซ้ายและขวา แต่ละวงมีจุดที่เคลื่อนที่ได้ซึ่งติดตามตำแหน่งแกนอะนาล็อกแบบ real-time ตารางสถานะปุ่มแสดงปุ่ม gamepad ทั้ง 12 ปุ่มเป็นตัวบ่งชี้แบบ LED ที่สว่างเป็นสีเขียวเมื่อกด

ตัวแสดงผลสติ๊กใช้ LVGL object ที่วางตำแหน่งภายในคอนเทนเนอร์วงกลม ตำแหน่งจุดถูกแมปจากช่วงแกน 0-255 ไปยังรัศมีวงกลม โดย 128 (กลาง) วางจุดที่ศูนย์กลางวงกลม การแมปรักษา deadzone แบบวงกลมและขอบเขตสูงสุด ให้การแสดงผลที่ตรงกับการเคลื่อนที่จริงของสติ๊ก

แต่ละวงกลมสติ๊กมีเส้น crosshair อ้างอิงที่ศูนย์กลาง และค่าตัวเลขดิบของแกน (0-255) แสดงด้านล่าง ตารางปุ่มจัดเรียงปุ่ม F310 ทั้ง 12 ปุ่ม (X, A, B, Y, LB, RB, LT, RT, Back, Start, L3, R3) ในเลย์เอาต์ grid โดยใช้สีพื้นหลังเขียว/เทาแสดงสถานะกด/ปล่อย

ส่วนเพิ่มเติมแสดง D-pad hat switch เป็น compass rose พร้อมตัวบ่งชี้ 8 ทิศทาง ทิศทางที่ใช้งานจะสว่างขึ้นขณะทิศอื่นยังคงหรี่ การแสดงผลแบบครบวงจรนี้มีประโยชน์สำหรับการปรับเทียบอินพุต joystick, ทดสอบการแมปปุ่ม และดีบัก pipeline จาก USB HID ไปยัง IPC bridge
