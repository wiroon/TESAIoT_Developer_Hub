# หน้าปัดนาฬิกาอัจฉริยะ (Smart Watch Face)

ตัวอย่างนี้สร้างหน้าจอแบบนาฬิกา พร้อมตัวเลขเวลาดิจิทัลขนาดใหญ่, arc แสดงวินาทีแบบหมุน และการ์ดสรุปเซ็นเซอร์ขนาดกะทัดรัดจัดเรียงรอบหน้าปัดนาฬิกา หน้าจออัปเดตทุกวินาทีผ่าน LVGL timer

หน้าปัดนาฬิกามีการแสดงชั่วโมง:นาที ขนาดใหญ่เด่นชัดตรงกลางด้วยฟอนต์ขนาดใหญ่ ล้อมรอบด้วย arc วินาทีแบบวงกลมที่กวาดจาก 0 ถึง 360 องศาตลอด 60 วินาที ค่าวินาทีปัจจุบันแสดงใต้เวลา เนื่องจากบอร์ด PSoC Edge ไม่มีแบตเตอรี่สำรอง RTC เวลาจึงคำนวณจาก FreeRTOS tick count ตั้งแต่เปิดเครื่อง ให้นาฬิกาแบบ uptime

การ์ดสรุปเซ็นเซอร์ 4 ใบจัดเรียงใต้หน้าปัดนาฬิกา แสดงค่าอ่านล่าสุดจากเซ็นเซอร์ที่มี แต่ละการ์ดแสดงไอคอนเซ็นเซอร์, ชื่อ และค่าปัจจุบัน การ์ดใช้ BSP feature flag guard เพื่อให้หน้าจอปรับตัวอัตโนมัติตามบอร์ดที่เชื่อมต่อ -- AI Kit แสดงอุณหภูมิ, ความดัน, ความชื้น และทิศเข็มทิศ ขณะที่ Eva Kit แสดงเซ็นเซอร์ที่มี

ตัวอย่างนี้สาธิตเทคนิคเลย์เอาต์ UI แบบวงกลม, แอนิเมชันที่ขับเคลื่อนด้วย timer, การผสานเซ็นเซอร์หลายตัวด้วย conditional compilation และรูปแบบการออกแบบหน้าปัดนาฬิกาที่เป็นประโยชน์สำหรับแอปพลิเคชัน wearable หรือแสดงสถานะ

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit (การ์ดเซ็นเซอร์ปรับตามบอร์ด)
