# สลับแหล่งกล้อง (Camera Source Switch)

ตัวอย่างนี้สาธิตการสลับขณะทำงานระหว่างอินเทอร์เฟซกล้อง DVP แบบขนาน (เซ็นเซอร์ OV7675) กับกล้อง USB UVC ปุ่มสลับช่วยให้ผู้ใช้เปลี่ยนแหล่งกล้องได้โดยไม่ต้องรีสตาร์ทแอปพลิเคชัน พร้อมแสดงแหล่งกล้องปัจจุบัน ความละเอียด และอัตราเฟรมแบบเรียลไทม์

ชั้น Camera Manager ให้ API แบบรวมสำหรับกล้องทั้งสองประเภท การสลับเกี่ยวข้องกับการหยุดแหล่งปัจจุบัน ปรับค่า Pipeline ใหม่ และเริ่มแหล่งใหม่ การเปลี่ยนใช้เวลาประมาณ 500ms สำหรับ DVP และ 1-2 วินาทีสำหรับ USB (เนื่องจากการค้นหาอุปกรณ์)

UI แสดงการ์ดสถานะที่มีแหล่งกล้องที่ใช้งาน ความละเอียดปัจจุบัน รูปแบบสี และ FPS ที่วัดได้ LVGL Canvas แสดงตัวอย่างภาพสดจากกล้องที่ใช้งานอยู่ การจัดการข้อผิดพลาดครอบคลุมกรณีที่กล้อง USB ไม่ได้เชื่อมต่อทางกายภาพ

รูปแบบนี้มีประโยชน์สำหรับผลิตภัณฑ์ที่รองรับเซ็นเซอร์ภาพหลายตัว เช่น อุปกรณ์ที่มีเซ็นเซอร์ DVP ในตัวสำหรับการทำงานตลอดเวลา และพอร์ต USB สำหรับกล้องภายนอกคุณภาพสูง Camera Manager รับประกันว่ามีเพียงแหล่งเดียวที่ทำงานในเวลาเดียวกันเพื่อหลีกเลี่ยงการแย่งทรัพยากร

รองรับ TESAIoT Dev Kit และ PSoC Edge AI Kit เท่านั้น เนื่องจากมีทั้งอินเทอร์เฟซกล้อง DVP และพอร์ต USB Type-A Host
