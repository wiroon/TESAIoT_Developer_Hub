# ตรวจจับการมีอยู่ด้วยเรดาร์ (Radar Presence Detection)

ตัวอย่างนี้สาธิตการตรวจจับการมีอยู่แบบ real-time โดยใช้เซ็นเซอร์เรดาร์ Infineon BGT60TR13C 60 GHz โมดูลเรดาร์ทำการวัด continuous wave Doppler เพื่อตรวจจับการมีอยู่ของมนุษย์ภายในมุมมอง และตัวอย่างนี้แสดงผลสถานะการตรวจจับด้วยหน้าจอเรดาร์แบบกวาดหมุนพร้อมแอนิเมชัน

การแสดงผลประกอบด้วยองค์ประกอบหลัก 3 ส่วน ได้แก่ arc กวาดหมุนที่จำลองหน้าจอเรดาร์คลาสสิก, ตัวบ่งชี้ขนาดใหญ่ตรงกลางที่เปลี่ยนสีตามสถานะการตรวจจับ (เขียวเมื่อตรวจพบ, เทาเมื่อไม่พบ) และ bar แสดงระยะโดยประมาณถึงเป้าหมายที่ตรวจพบ arc กวาดหมุนอย่างต่อเนื่องผ่าน LVGL timer สร้างแอนิเมชันการสแกนเรดาร์ที่คุ้นเคย

ข้อมูลการมีอยู่จากเรดาร์อ่านจาก BGT60TR13C ผ่าน radar task API เซ็นเซอร์ส่งออกค่า flag การมีอยู่/ไม่มีอยู่แบบไบนารี พร้อมค่าระยะโดยประมาณ ตัวอย่างสาธิตการเริ่มต้นเซ็นเซอร์อย่างถูกต้อง, การ polling เป็นระยะในอัตราที่เหมาะสม (200ms) และการแมปข้อมูลเซ็นเซอร์กับองค์ประกอบภาพ

ตัวอย่างนี้สาธิตการผสานเซ็นเซอร์เรดาร์, องค์ประกอบ UI แบบมีแอนิเมชัน และรูปแบบการแสดงผลเซ็นเซอร์แบบ real-time ต้องใช้บอร์ด AI Kit ซึ่งมีเซ็นเซอร์เรดาร์ BGT60TR13C
