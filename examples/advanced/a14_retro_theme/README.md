# ธีม Retro Game Boy (Retro Game Boy Theme)

ตัวอย่างนี้สร้างธีมภาพแรงบันดาลใจจาก Game Boy โดยใช้จานสีเขียว 4 โทนคลาสสิก, เอฟเฟกต์เส้นสแกน CRT ที่เรนเดอร์บน LVGL canvas และป้าย "DOT MATRIX WITH STEREO SOUND" อันเป็นเอกลักษณ์ ทำหน้าที่เป็นโมดูลธีมที่ใช้ซ้ำได้สำหรับตัวอย่างเกม

จานสี Game Boy ประกอบด้วย 4 เฉดสีเฉพาะ ได้แก่ สว่างที่สุด (#9BBC0F), สว่าง (#8BAC0F), เข้ม (#306230) และเข้มที่สุด (#0F380F) สีเหล่านี้ถูกนำไปใช้กับ LVGL widget เพื่อสร้างสุนทรียะ retro ที่สมจริง พื้นหลังใช้เฉดเข้มที่สุด ข้อความใช้เฉดสว่างที่สุด และองค์ประกอบ UI ใช้โทนกลาง

เอฟเฟกต์เส้นสแกน CRT ทำได้โดยการวาดเส้นแนวนอนสลับกันด้วยความทึบที่ลดลงบน canvas overlay ทุกแถวพิกเซลเว้นแถวจะถูกทำให้เข้มขึ้นเพื่อจำลองรูปลักษณ์ของจอ CRT เก่า overlay นี้เรนเดอร์ครั้งเดียวและวางทับเนื้อหาเกมด้วยความโปร่งใสบางส่วน

ป้าย "DOT MATRIX" จำลองกรอบหน้าจอ Game Boy ต้นฉบับ ใช้คอนเทนเนอร์ที่มีขอบจุด, พื้นหลังเขียวเข้มที่สุด และข้อความเฉพาะตัวในสีเขียวสว่างที่สุด widget ตัวอย่างสาธิตว่าสีจานสีแต่ละสีถูกแมปกับองค์ประกอบ UI ทั่วไปอย่างไร เช่น ปุ่ม, label, bar และแผง

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
