# โรงงานสร้างการ์ด (Card Factory)

ตัวอย่างนี้ให้ฟังก์ชัน `tesaiot_card_create()` ที่นำกลับมาใช้ได้ สำหรับสร้างคอนเทนเนอร์การ์ดที่จัดสไตล์ตามระบบออกแบบ BENTO ฟังก์ชันรับ parent, ข้อความชื่อ, สีเน้น, ความกว้าง และความสูง คืนค่าคอนเทนเนอร์พร้อมสำหรับใส่ widget เนื้อหา

แต่ละการ์ดที่ผลิตจากโรงงานมีพื้นหลังสีเข้ม (0x142240) มุมโค้งมน (radius 12) ขอบสีเน้น 1 พิกเซล และเงาบาง label ชื่อวางที่มุมซ้ายบนของการ์ดด้วยสีเน้น เส้นแบ่งแนวนอนแยกชื่อจากพื้นที่เนื้อหา

ตัวอย่างสร้างการ์ดตัวอย่าง 4 ใบที่มีขนาดและสีต่างกัน จัดเรียงแบบ 2x2 สาธิตว่าฟังก์ชัน factory ขจัดโค้ดจัดสไตล์ซ้ำซ้อน แต่ละการ์ดมีเนื้อหา placeholder (label ค่า) เพื่อแสดงตำแหน่งพื้นที่เนื้อหา

รูปแบบ factory นี้ใช้ทั่ว firmware จริงที่ต้องการการ์ดเซ็นเซอร์ แผง dashboard หรือไทล์ข้อมูล การรวมสไตล์การ์ดไว้ในฟังก์ชันเดียวทำให้มั่นใจในความสม่ำเสมอทางภาพ และทำให้การเปลี่ยนธีมเป็นการแก้ไขจุดเดียว

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
