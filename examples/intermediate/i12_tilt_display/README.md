# แสดงมุมเอียง (Tilt Display)

ตัวอย่างนี้คำนวณมุม roll และ pitch จากข้อมูล BMI270 accelerometer โดยใช้ `atan2f` และแสดงเป็น arc gauge 2 ตัววางเคียงกัน Roll วัดการหมุนรอบแกน X และ pitch วัดการหมุนรอบแกน Y ทั้งคู่ในหน่วยองศาจาก -90 ถึง +90

มุม roll คำนวณจาก `atan2f(accel_y, accel_z)` และ pitch จาก `atan2f(-accel_x, sqrtf(accel_y^2 + accel_z^2))` สูตรมาตรฐานด้านการบินเหล่านี้แปลงเวกเตอร์ accelerometer ดิบเป็นมุมเอียงที่เข้าใจง่าย การคำนวณนี้สมมติว่าบอร์ดไม่อยู่ในสภาวะตกอิสระ

arc gauge แต่ละตัวครอบคลุมจาก -90 ถึง +90 องศาด้วยการกวาดจาก 135 ถึง 45 องศา arc roll ใช้สีเขียวและ arc pitch ใช้สีส้ม ตรงกับแบบแผนสีของ dashboard ดั้งเดิม label ตรงกลางแสดงมุมเป็นตัวเลข และ label ชื่อระบุ gauge แต่ละตัว

timer ที่ทำงานทุก 50ms ให้การตอบสนองภาพที่ลื่นไหลเมื่อเอียงบอร์ด วิธีนี้ทำงานได้ดีสำหรับการวัดเอียงแบบหยุดนิ่งหรือเคลื่อนที่ช้า สำหรับการใช้งานแบบไดนามิก (หมุนเร็ว) จำเป็นต้องใช้ gyroscope integration ร่วมกับ complementary filter (ครอบคลุมในตัวอย่างขั้นสูง)
