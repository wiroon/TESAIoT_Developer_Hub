# การ์ดเซ็นเซอร์สภาพแวดล้อม (Environmental Cards)

ตัวอย่างนี้แสดงค่าเซ็นเซอร์สภาพแวดล้อม 3 ค่าเป็น arc gauge ภายในการ์ดที่จัดสไตล์ อุณหภูมิและความดันมาจากเซ็นเซอร์ DPS368 ส่วนความชื้นมาจากเซ็นเซอร์ SHT40 เซ็นเซอร์ทั้งสองมีเฉพาะใน AI Kit ดังนั้นตัวอย่างนี้ถูกครอบด้วย `#if BSP_HAS_DPS368 && BSP_HAS_SHT40`

แต่ละการ์ดเป็นคอนเทนเนอร์โค้งมน (radius 12) พร้อมพื้นหลัง BENTO dark theme ภายในแต่ละการ์ดมี arc gauge ที่เติมจาก 0 ถึงค่าสูงสุดที่เหมาะสมกับการวัด: 0-50 สำหรับอุณหภูมิ (เซลเซียส) 900-1100 สำหรับความดัน (hPa) และ 0-100 สำหรับความชื้น (%) label ตรงกลางแสดงค่าตัวเลข

สี arc เป็นไปตามธีม dashboard ดั้งเดิม: ส้ม (0xFF9800) สำหรับอุณหภูมิ น้ำเงิน (0x2196F3) สำหรับความชื้น และฟ้าสำหรับความดัน สีเหล่านี้ตรงกับ dashboard จริงอย่างแม่นยำ รักษาความสม่ำเสมอทางภาพ

timer 1 วินาทีอ่านเซ็นเซอร์ทั้งสองและอัปเดต arc และ label ค่าทั้งหมด ข้อมูลสภาพแวดล้อมเปลี่ยนช้า ดังนั้นช่วงเวลาที่ยาวกว่าจึงเหมาะสมและลดภาระ CPU เทียบกับ 50ms ที่ใช้สำหรับ IMU
