# กราฟแกน Y คู่ (Dual-Axis Chart)

ตัวอย่างนี้สาธิตกราฟแกน Y คู่โดยพล็อตอุณหภูมิบนแกนหลักและความดันบรรยากาศบนแกนรอง ทั้งสองจากเซ็นเซอร์ DPS368 ตัวอย่างนี้ใช้ได้เฉพาะ AI Kit เนื่องจาก Eva Kit ไม่มีเซ็นเซอร์ DPS368

สร้าง 2 series บนแกนต่างกัน: อุณหภูมิเป็นองศาเซลเซียสบน `LV_CHART_AXIS_PRIMARY_Y` (ช่วง 15-45) และความดันเป็น hPa บน `LV_CHART_AXIS_SECONDARY_Y` (ช่วง 950-1050) series สีส้มแสดงอุณหภูมิ ส่วน series สีฟ้าแสดงความดัน แต่ละตัวสเกลตามแกนของตัวเอง

ตัวอย่างทั้งหมดถูกครอบด้วย `#if BSP_HAS_DPS368` เพื่อให้คอมไพล์ได้สะอาดบนบอร์ดที่ไม่มีเซ็นเซอร์ barometric เมื่อ guard ประเมินเป็น false ตัวอย่างจะแสดงข้อความ "not available" แทน

timer ที่ทำงานทุก 500ms อ่านทั้งอุณหภูมิและความดันจาก DPS368 และส่งค่าเข้า chart series ที่เกี่ยวข้อง อัตราอัปเดตที่ช้ากว่านี้เหมาะสมสำหรับเซ็นเซอร์สภาพแวดล้อมที่เปลี่ยนแปลงช้า label ค่า 2 ตัวใต้กราฟแสดงค่าปัจจุบันพร้อมหน่วย
