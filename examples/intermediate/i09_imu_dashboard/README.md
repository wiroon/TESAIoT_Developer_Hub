# แดชบอร์ด IMU (IMU Dashboard)

ตัวอย่างนี้สร้างแดชบอร์ด IMU 6 แกนเต็มรูปแบบ รวมกราฟ 2 ตัวและ label ค่าตัวเลข 6 ตัวในเลย์เอาต์แบบการ์ด การ์ดบนมีกราฟ accelerometer 3 series และการ์ดล่างมีกราฟ gyroscope 3 series ทั้งสองอัปเดตทุก 50ms จากเซ็นเซอร์ BMI270

แต่ละการ์ดเป็นคอนเทนเนอร์ที่จัดสไตล์ด้วยพื้นหลังสีเข้ม (0x142240) มุมโค้งมน (radius 12) ขอบบาง และ label ชื่อ ภายในแต่ละการ์ด กราฟครอบครองพื้นที่หลักพร้อม label ค่า 3 ตัว (X/Y/Z) วางไว้ด้านล่าง รูปแบบ card factory นี้เป็นองค์ประกอบ dashboard มาตรฐานที่ใช้ทั่ว firmware

กราฟ accelerometer แสดงค่าเป็น milli-g (ช่วง -2000 ถึง +2000) ขณะที่กราฟ gyroscope แสดงองศาต่อวินาที (ช่วง -500 ถึง +500) ทั้งสองใช้โทนสีแดง/เขียว/น้ำเงินเหมือนกันสำหรับ X/Y/Z เพื่อความสม่ำเสมอข้ามแพลตฟอร์ม

LVGL timer ตัวเดียวจัดการทั้งการอ่านเซ็นเซอร์และการอัปเดตกราฟ ทำให้ข้อมูลซิงโครไนซ์ระหว่าง 2 จอแสดงผล รูปแบบแดชบอร์ดนี้ขยายได้สำหรับการ์ดเซ็นเซอร์เพิ่มเติม และเป็นพื้นฐานของหน้า sensor hub จริง

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
