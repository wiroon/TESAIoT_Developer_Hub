# เข็มทิศแสดงทิศทาง (Compass Heading)

ตัวอย่างนี้อ่าน BMM350 magnetometer และคำนวณมุมทิศทาง 0 ถึง 360 องศา แสดงเป็น arc gauge วงกลมเต็มพร้อมข้อความทิศหลัก (N, NE, E, SE, S, SW, W, NW) ตรงกลาง

มุมทิศทางคำนวณด้วย `atan2f(mag_y, mag_x)` และแปลงจากเรเดียนเป็นองศา ค่าลบถูกปรับให้อยู่ในช่วง 0-360 ผลลัพธ์ขับเคลื่อน arc widget ที่กำหนดค่าเป็นตัวบ่งชี้ 360 องศาเต็ม

label ตรงกลางขนาดใหญ่แสดงค่าทิศทางเป็นตัวเลขในหน่วยองศา และ label รองด้านล่างแสดงทิศหลักเป็นข้อความ ทิศหลักกำหนดโดยแบ่งมุมเป็น 8 ส่วนส่วนละ 45 องศา

arc ใช้สีตัวบ่งชี้ม่วงแดง (0xE040FB) ตรงกับสีธีม BMM350 จาก dashboard ดั้งเดิม พื้นหลัง arc เป็นสีน้ำเงินเข้ม ตัวบ่งชี้กวาดจาก 0 (ทิศเหนือ) ตามเข็มนาฬิกา arc อัปเดตทุก 100ms เพื่อพฤติกรรมเข็มทิศที่ตอบสนองไว
