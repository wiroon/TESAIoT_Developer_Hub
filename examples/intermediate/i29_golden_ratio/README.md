# เลย์เอาต์อัตราส่วนทอง (Golden Ratio Layout)

ตัวอย่างนี้สาธิตการจัดวาง UI แบบสัดส่วนโดยใช้อัตราส่วนทอง หน้าจอถูกแบ่งเป็น 2 คอลัมน์: คอลัมน์หลัก (กว้าง 61.8%) และคอลัมน์รอง (กว้าง 38.2%) สร้างเลย์เอาต์ที่สมดุลและสวยงาม

ค่าคงที่ `UI_PHI_MAJOR` (0.618) และ `UI_PHI_MINOR` (0.382) ใช้คำนวณความกว้างคอลัมน์จากความกว้าง parent ที่มี สัดส่วนเหล่านี้มาจากอัตราส่วนทอง (1.618) และใช้กันอย่างแพร่หลายในงานออกแบบสำหรับการแบ่งส่วนที่ดูเป็นธรรมชาติ

คอลัมน์หลักเก็บเนื้อหาหลัก (กราฟ) ขณะที่คอลัมน์รองเก็บข้อมูลสนับสนุน (label ค่า, ตัวบ่งชี้สถานะ) เลย์เอาต์นี้ดึงดูดสายตาไปยังพื้นที่ใหญ่กว่าโดยธรรมชาติ ขณะที่เก็บข้อมูลเสริมไว้ให้เข้าถึงได้

อัตราส่วนทองสามารถใช้แบบ recursive ได้: คอลัมน์รองเองก็สามารถแบ่งด้วยสัดส่วนเดียวกันสำหรับส่วนย่อย ตัวอย่างนี้แสดงการแบ่งครั้งเดียวเพื่อความชัดเจน แต่รูปแบบขยายไปยังเลย์เอาต์ซ้อนทั่วการออกแบบ BENTO dashboard
