# ตรวจจับใบหน้าด้วย ML (Face Detection with ML)

ตัวอย่างนี้รันการตรวจจับใบหน้าแบบเรียลไทม์โดยใช้โมเดลโครงข่ายประสาทเทียม SSD Nano บนเฟรมจากกล้อง USB UVC ใบหน้าที่ตรวจพบจะถูกเน้นด้วยกรอบสี่เหลี่ยมซ้อนทับบน LVGL Canvas พร้อมเปอร์เซ็นต์ความมั่นใจ จำนวนใบหน้า และเวลาในการอนุมาน

Pipeline การอนุมาน ML ประกอบด้วย 3 ขั้นตอน: จับภาพจาก Camera Manager, ประมวลผลเบื้องต้น (ปรับขนาดและ Normalize ให้ตรงกับมิติอินพุตของโมเดล) และอนุมานโดยใช้เอนจินตรวจจับใบหน้า โมเดล SSD Nano ทำงานบน CM55 Core ซึ่งมีฮาร์ดแวร์รองรับการเร่งโครงข่ายประสาทเทียม การอนุมานแต่ละครั้งจะสร้างรายการ Bounding Box พร้อมคะแนนความมั่นใจ

LVGL Canvas แสดงตัวอย่างภาพจากกล้องพร้อม Bounding Box ซ้อนทับ ใบหน้าที่ตรวจพบแต่ละใบหน้าจะได้กรอบสี่เหลี่ยมสีเขียวพร้อมเปอร์เซ็นต์ความมั่นใจแสดงอยู่ด้านบน การวาด Overlay ใช้ฟังก์ชัน LVGL Canvas Draw (lv_canvas_draw_rect) หลังจากอัปเดตเฟรมแต่ละครั้ง แผงข้อมูลแยกต่างหากแสดงจำนวนใบหน้ารวม เวลาอนุมานเฉลี่ยเป็นมิลลิวินาที และ FPS ปัจจุบัน

การปรับแต่งประสิทธิภาพเป็นสิ่งสำคัญ: อินพุตของโมเดลปกติเป็น 160x160 หรือ 128x128 ขณะที่ Canvas แสดงผลที่ 320x240 เฟรมถูกย่อขนาดสำหรับการอนุมานแต่แสดงที่ความละเอียดเต็ม เวลาอนุมานแตกต่างจาก 30-80ms ขึ้นอยู่กับความซับซ้อนของฉาก ดังนั้น FPS ที่ได้จริงเมื่อเปิดการตรวจจับคือประมาณ 10-15 FPS

รองรับ TESAIoT Dev Kit และ PSoC Edge AI Kit เท่านั้น เนื่องจากต้องใช้กล้อง USB, เอนจินโครงข่ายประสาทเทียม CM55 และ PSRAM เพียงพอสำหรับ Frame Buffer และน้ำหนักโมเดล
