# เกมงู (Snake Game)

ตัวอย่างนี้สร้างเกมงูแบบคลาสสิกบนกริด 20x15 ช่อง โดยใช้ LVGL Canvas สำหรับการเรนเดอร์ งูควบคุมได้ผ่านจอยสติ๊ก USB F310 (ผ่าน IPC Bridge) หรือ D-pad แบบสัมผัสบนหน้าจอ เกมประกอบด้วยการเก็บอาหาร การยาวขึ้นของงู การตรวจจับชนกำแพงและตัวเอง และคะแนนสูงสุดแบบถาวร

ตัวงูถูกจัดการเป็น Circular Buffer ของพิกัดกริด ซึ่งให้การเคลื่อนที่แบบ O(1) ในแต่ละ Game Tick หัวจะเคลื่อนไปในทิศทางปัจจุบันและหางจะถูกลบ (ยกเว้นเมื่องูเพิ่งกินอาหาร) การตรวจจับการชนจะตรวจสอบตำแหน่งหัวกับกำแพงและส่วนตัวทั้งหมด

อาหารเกิดขึ้นที่ตำแหน่งกริดว่างแบบสุ่มโดยใช้ PRNG ที่ใช้ System Tick Counter เป็นค่าเริ่มต้น เมื่องูกินอาหารจะยาวขึ้น 1 ส่วน คะแนนเพิ่มขึ้น และความเร็วเกมเพิ่มขึ้นเล็กน้อย (Tick Interval ลดลง 5ms ต่ำสุด 80ms)

การเรนเดอร์ใช้ LVGL Canvas พร้อมการจัดการพิกเซลโดยตรงสำหรับกริด แต่ละช่องถูกวาดเป็นสี่เหลี่ยมสี: สีเขียวสำหรับตัวงู สีเขียวสว่างสำหรับหัว สีแดงสำหรับอาหาร และสีน้ำเงินเข้มสำหรับพื้นหลัง หน้าจอจบเกมแสดงคะแนนสุดท้ายและปุ่มเริ่มใหม่
