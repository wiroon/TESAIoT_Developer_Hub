# แสดงภาพจากกล้องแบบเรียลไทม์ (Camera Preview)

ตัวอย่างนี้จับภาพจากกล้อง USB UVC และแสดงบน LVGL Canvas แบบเรียลไทม์ Canvas ตั้งค่าที่ความละเอียด 320x240 ในรูปแบบสี RGB565 ซึ่งตรงกับเอาต์พุตดั้งเดิมของกล้อง ตัวนับ FPS และปุ่มเริ่ม/หยุดให้การควบคุมขณะทำงาน

ชั้น Camera Manager จัดการการเริ่มต้น USB Host, การค้นหาอุปกรณ์ และการจัดการ Frame Buffer เฟรมถูกส่งผ่าน Callback ที่คัดลอกข้อมูล RGB565 ลงใน Draw Buffer ของ LVGL Canvas วิดเจ็ต LVGL Canvas ให้การเข้าถึง Pixel Buffer โดยตรง หลีกเลี่ยงการแปลงรูปแบบที่สิ้นเปลือง

FreeRTOS Task แยกต่างหากจะดึงเฟรมใหม่จาก Camera Manager ที่อัตราสูงสุด 30 FPS โดย Task ใช้ lv_lock/lv_unlock เพื่ออัปเดต Canvas Buffer อย่างปลอดภัยจากนอกบริบทการรีเฟรชหน้าจอ LVGL ตัวนับ FPS ใช้หน้าต่างเลื่อนของ Timestamp 10 เฟรมล่าสุดเพื่อคำนวณค่าเฉลี่ยที่เสถียร

การจัดการหน่วยความจำเป็นสิ่งสำคัญ: Canvas Buffer (320x240x2 = 150 KB) ถูกจัดสรรจาก PSRAM เพื่อหลีกเลี่ยงการใช้ SRAM ภายในที่มีจำกัด Frame Buffer ของกล้องถูกจัดการโดย USB Host Stack และต้องไม่ถูกเก็บค้างข้ามขอบเขตเฟรม

รองรับ TESAIoT Dev Kit และ PSoC Edge AI Kit เท่านั้น เนื่องจากต้องใช้พอร์ต USB Type-A Host สำหรับเชื่อมต่อกล้อง UVC
