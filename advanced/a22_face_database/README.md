# ฐานข้อมูลใบหน้า (Face Database)

ตัวอย่างนี้สร้างระบบ UI สำหรับลงทะเบียนและจดจำใบหน้า (Face ID) มี gallery grid แสดงภาพขนาดย่อของใบหน้าที่ลงทะเบียนแล้ว, ตัวควบคุมสำหรับลงทะเบียนใบหน้าใหม่และเรียกใช้การจดจำ และแผงแสดงผลลัพธ์ที่แสดงคะแนนความเชื่อมั่นและตัวตน

ขั้นตอนการลงทะเบียนจับเฟรมจากกล้อง, รัน face detection เพื่อหาใบหน้าในภาพ, แยก face embedding vector และเก็บลงฐานข้อมูลใบหน้าพร้อมชื่อที่ผู้ใช้กำหนด gallery grid อัปเดตอัตโนมัติเพื่อแสดงภาพขนาดย่อของใบหน้าที่ลงทะเบียนแต่ละรายการ สามารถเก็บใบหน้าได้สูงสุด 10 รายการ

โหมดจดจำจับเฟรม, ตรวจจับใบหน้า, คำนวณ embedding และเปรียบเทียบกับ embedding ที่ลงทะเบียนทั้งหมดโดยใช้ cosine similarity ผลลัพธ์ที่ตรงที่สุดแสดงพร้อมเปอร์เซ็นต์ความเชื่อมั่นและชื่อที่ลงทะเบียน เกณฑ์ความเชื่อมั่น 70% กำหนดว่าจะยอมรับหรือปฏิเสธการจับคู่ ผลลัพธ์แสดงเป็นตัวบ่งชี้สีเขียว (ตรงกัน) หรือแดง (ไม่ตรง)

เลย์เอาต์ UI ประกอบด้วย 3 ส่วน ได้แก่ พื้นที่ preview กล้องด้านบนแสดงเฟรมสดหรือเฟรมที่จับพร้อมกรอบ bounding box ตรวจจับใบหน้าซ้อนทับ, gallery grid ตรงกลางแสดงภาพขนาดย่อใบหน้าที่ลงทะเบียนในแถวเลื่อนได้ และแผงควบคุมด้านล่างพร้อมปุ่ม Enroll, Recognize และ Delete รวมถึงหน้าจอแสดงผลลัพธ์

ตัวอย่างนี้สาธิตการผสาน AI inference, pipeline ตรวจจับและจดจำใบหน้า, การจัดการภาพขนาดย่อ, การดำเนินการฐานข้อมูล และรูปแบบ UI แบบโต้ตอบสำหรับแอปพลิเคชัน biometric
