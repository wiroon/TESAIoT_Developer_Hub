# แดชบอร์ดสำหรับงานผลิต (Production Dashboard)

ตัวอย่างนี้สร้างแดชบอร์ดระดับอุตสาหกรรมที่แสดงข้อมูลแบบ real-time จากเซ็นเซอร์ BSP ทั้งหมดที่มี พร้อม compile-time feature flag guard ที่ถูกต้อง แดชบอร์ดปรับเลย์เอาต์อัตโนมัติตามบอร์ดที่เชื่อมต่อ แสดงเฉพาะเซ็นเซอร์ที่มีอยู่จริงบนบอร์ด

เลย์เอาต์ประกอบด้วยการ์ดเซ็นเซอร์ 4 ใบจัดเรียงแบบ grid 2x2 ด้านบน, กราฟเส้นแบบเลื่อนตรงกลางที่ติดตามค่าวัดหลักตลอดเวลา และแถบตัวบ่งชี้สถานะด้านล่างแสดงสุขภาพระบบ แต่ละการ์ดเซ็นเซอร์แสดงชื่อเซ็นเซอร์, ค่าปัจจุบันพร้อมหน่วย และตัวบ่งชี้แนวโน้มขนาดเล็ก การ์ดใช้ชุดสี TESAIoT มาตรฐาน ได้แก่ BMI270 เขียว, DPS368 ส้ม, SHT40 น้ำเงิน, BMM350 ม่วง

การอ่านเซ็นเซอร์ทั้งหมดถูกห่อด้วย `BSP_HAS_*` preprocessor guard เพื่อให้โค้ด compile ได้สะอาดสำหรับทุกบอร์ด บน AI Kit การ์ดเซ็นเซอร์ทั้ง 4 ใบแสดง (IMU, barometer, humidity, magnetometer) บน Eva Kit การ์ด DPS368 และ SHT40 ถูกแทนที่ด้วยตัวบ่งชี้ CapSense และ potentiometer กราฟ plot ค่า IMU accelerometer magnitude เสมอ เนื่องจาก BMI270 มีอยู่บนทุกบอร์ด

ตัวอย่างนี้สาธิตรูปแบบคุณภาพระดับ production ได้แก่ conditional compilation สำหรับตัวแปรฮาร์ดแวร์, การ polling เซ็นเซอร์หลายตัวด้วย timer เดียว, การจัดการข้อมูลกราฟพร้อมการเลื่อนอัตโนมัติ และการรวมสถานะ เป็น template เริ่มต้นสำหรับแอปพลิเคชันตรวจสอบอุตสาหกรรมจริง

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit (ชุดเซ็นเซอร์ปรับตามบอร์ด)
