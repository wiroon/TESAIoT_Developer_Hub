# สแกน WiFi แสดงผลบนตาราง (WiFi Scan + Table Display)

ตัวอย่างนี้สาธิตการสแกนหา Access Point ของ WiFi โดยใช้ Cypress WiFi Connection Manager (WCM) และแสดงผลลัพธ์ในวิดเจ็ตตาราง LVGL แบบเรียงลำดับ การสแกนทำงานแบบ Asynchronous ผ่าน Callback โดยเก็บ Access Point ที่ไม่ซ้ำกันได้สูงสุด 20 รายการ พร้อมข้อมูล SSID, RSSI, ช่องสัญญาณ และประเภทการเข้ารหัส

Callback ของการสแกนจะกรอง SSID ที่ซ้ำกัน และเรียงลำดับตามความแรงของสัญญาณ (RSSI จากมากไปน้อย) ทำให้เครือข่ายที่สัญญาณแรงที่สุดแสดงอยู่ด้านบน วิดเจ็ต Spinner ให้ข้อมูลสถานะระหว่างการสแกน ซึ่งโดยปกติจะเสร็จภายใน 3-5 วินาที ขึ้นอยู่กับสภาพแวดล้อม RF

เมื่อการสแกนเสร็จสิ้น ผลลัพธ์จะถูกใส่ลงในตาราง LVGL ที่มี 4 คอลัมน์ ค่า RSSI จะแสดงสีตามความแรง: สีเขียวสำหรับสัญญาณแรง (ดีกว่า -50 dBm), สีเหลืองสำหรับสัญญาณปานกลาง (-50 ถึง -70 dBm) และสีแดงสำหรับสัญญาณอ่อน (ต่ำกว่า -70 dBm) คอลัมน์ประเภทการเข้ารหัสแสดงชื่อที่อ่านง่าย เช่น WPA2, WPA3, Open และ WEP

ปุ่มสแกนซ้ำด้านล่างช่วยให้ผู้ใช้สามารถสแกนเพิ่มเติมได้โดยไม่ต้องรีสตาร์ทแอปพลิเคชัน กระบวนการสแกนและแสดงผลทั้งหมดทำงานใน FreeRTOS Task เดียว โดยใช้ Semaphore เพื่อประสานงานระหว่าง WCM Callback กับโค้ดอัปเดต UI

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit เนื่องจาก WiFi ทำงานบน CM33_NS Core ผ่านชิป CYW55513
