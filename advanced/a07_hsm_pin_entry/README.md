# แผงกด PIN ด้วย HSM (HSM PIN Entry Pad)

ตัวอย่างนี้สร้างอินเทอร์เฟซกด PIN ที่ปลอดภัย ประกอบด้วยแป้นพิมพ์ตัวเลข 3x4 การแสดง PIN แบบปิดบัง การตรวจสอบด้วย SHA-256 Hash ผ่าน OPTIGA Trust M และกลไกล็อกหลังป้อนผิด 3 ครั้ง ตัวอย่างนี้แสดงรูปแบบการพิสูจน์ตัวตนที่พร้อมใช้งานจริง เหมาะสำหรับอุปกรณ์ฝังตัวที่มี Hardware Security Module

แป้นพิมพ์สร้างจาก LVGL Button Matrix ขนาด 3x4: ตัวเลข 1-9 ใน 3 แถวบน ปุ่มลบทางซ้าย เลข 0 ตรงกลาง และปุ่มยืนยันทางขวา การกดแต่ละครั้งจะเพิ่มตัวเลขลงในบัฟเฟอร์ PIN (สูงสุด 8 หลัก) และอัปเดตการแสดงผลแบบปิดบังที่แสดงจุดสำหรับแต่ละหลักที่ป้อน ปุ่มลบจะลบหลักสุดท้าย

เมื่อผู้ใช้ยืนยัน PIN ค่า SHA-256 Hash จะถูกคำนวณโดยใช้ OPTIGA Trust M และเปรียบเทียบกับ Hash อ้างอิงที่เก็บไว้ วิธีนี้ทำให้ PIN เป็นข้อความธรรมดาไม่เคยถูกเก็บใน Flash Memory การเข้าถึง OPTIGA ปฏิบัติตามโปรโตคอลการใช้บัส I2C ร่วมกันมาตรฐาน พร้อมคำสั่ง IPC หยุด/เริ่มหน้าจอสัมผัส

กลไกล็อกจะนับจำนวนครั้งที่ป้อนผิดและล็อกแป้นพิมพ์หลังป้อนผิดติดต่อกัน 3 ครั้ง ระหว่างล็อกจะแสดงตัวนับถอยหลัง (30 วินาที) และปิดการใช้งานปุ่มแป้นพิมพ์ทั้งหมด สิ่งนี้ป้องกันการโจมตีแบบ Brute-force กับ PIN ขนาด 4-8 หลัก ตัวนับจำนวนครั้งจะรีเซ็ตเมื่อพิสูจน์ตัวตนสำเร็จ
