# กราฟพร้อมคำอธิบาย (Chart with Legend)

ตัวอย่างนี้เพิ่มคำอธิบายสีใต้กราฟหลาย series แต่ละรายการคำอธิบายประกอบด้วยสี่เหลี่ยมสีเล็กๆ และ label ข้อความ จัดเรียงในแถว flex แนวนอน รูปแบบนี้ใช้ทั่ว BENTO dashboard สำหรับระบุ chart

สร้าง 3 data series ด้วยสีต่างกัน (แดง, เขียว, น้ำเงิน) และรายการคำอธิบายที่ตรงกันในคอนเทนเนอร์ flex ที่วางใต้กราฟ คอนเทนเนอร์คำอธิบายใช้ `lv_obj_set_flex_flow(LV_FLEX_FLOW_ROW)` พร้อมการจัดตำแหน่งกลางและระยะห่างระหว่างรายการ

แต่ละรายการคำอธิบายเป็นคอนเทนเนอร์เล็กๆ ที่มีสี่เหลี่ยมสี (16x16 พิกเซล) และ label สีพื้นหลังของสี่เหลี่ยมตรงกับสี series อย่างแม่นยำ สร้างความเชื่อมโยงภาพที่ชัดเจนระหว่างคำอธิบายและเส้นกราฟ

ข้อมูลกราฟขับเคลื่อนด้วยค่าสุ่มจาก timer เพื่อให้การแสดงผลเป็นไดนามิก ตัวอย่างนี้เน้นเทคนิคการจัดวาง UI มากกว่าการเชื่อมต่อเซ็นเซอร์ ทำให้เป็นรูปแบบที่นำกลับมาใช้ได้ซึ่งนักพัฒนาสามารถปรับใช้กับการแสดงผลหลาย series ใดๆ ก็ได้
