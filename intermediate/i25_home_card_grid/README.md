# Grid การ์ดหน้าหลัก (Home Card Grid)

ตัวอย่างนี้จำลองรูปแบบ grid การ์ดหน้า Home จาก `page_home.c` การ์ด 6 ใบจัดเรียงแบบ 3x2 แต่ละใบมีไอคอนและ label การแตะการ์ดทริกเกอร์ navigation callback จำลองการเรียก `pm_navigate` ที่จะโหลดหน้าที่เกี่ยวข้อง

คำจำกัดความการ์ดใช้ static array คล้าย `s_card_defs[]` ใน firmware จริง แต่ละรายการระบุ page ID, icon symbol, ชื่อที่แสดง และสีเน้น grid ถูกสร้างจาก array นี้แบบไดนามิก ดังนั้นการเพิ่มหรือลบการ์ดเป็นการแก้ไขบรรทัดเดียว

แต่ละการ์ดเป็นคอนเทนเนอร์ที่จัดสไตล์ด้วยพื้นหลังสีเข้ม มุมโค้งมน และขอบสีเน้น ไอคอนใช้ LVGL symbol (`LV_SYMBOL_*`) ที่ขนาดตัวอักษรใหญ่ และชื่อการ์ดแสดงด้านล่าง click event callback บนแต่ละการ์ดบันทึก page ID ที่ถูกแตะ

นี่คือตัวอย่าง dual-file sync: ใน firmware จริง ทุกการ์ดที่กำหนดที่นี่ต้องมี `pm_register` ที่ตรงกันใน `sensorhub_ui.c` การ์ดที่ไม่มีการลงทะเบียนจะ crash เมื่อแตะ การลงทะเบียนที่ไม่มีการ์ดทำให้หน้าเข้าถึงไม่ได้จาก Home
