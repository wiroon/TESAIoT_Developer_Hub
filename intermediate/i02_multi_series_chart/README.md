# กราฟหลาย Series (Multi-Series Chart)

ตัวอย่างนี้พล็อต 3 ชุดข้อมูลพร้อมกันบนกราฟเส้นเดียว แสดงแกน X, Y และ Z ของ BMI270 accelerometer แต่ละแกนมีสีกำหนดเฉพาะ: แดงสำหรับ X, เขียวสำหรับ Y และน้ำเงินสำหรับ Z ทำให้แยกแยะ 3 ช่องสัญญาณได้ง่าย

สร้าง 3 series ด้วย `lv_chart_add_series` แต่ละ series ผูกกับแกน Y หลักแต่มีสีต่างกัน กราฟเก็บข้อมูล 80 จุดต่อ series และช่วงแกน Y ครอบคลุมช่วงเอาต์พุต accelerometer ปกติที่ -2g ถึง +2g (สเกลเป็น -2000 ถึง +2000 milli-g)

LVGL timer ที่ทำงานทุก 50ms อ่านเซ็นเซอร์ BMI270 และส่งค่า X, Y, Z ล่าสุดเข้า series ที่เกี่ยวข้องผ่าน `lv_chart_set_next_value` label ค่าตัวเลข 3 ตัวใต้กราฟแสดงค่าแบบเรียลไทม์ โดยแต่ละตัวมีสีตรงกับ series ของมัน

พื้นหลังกราฟใช้สี BENTO dark theme (0x142240) พร้อมเส้น grid เพื่อความอ่านง่าย รูปแบบการซ้อน series หลายชุดบนกราฟเดียวนี้เป็นพื้นฐานของการแสดงผลเซ็นเซอร์หลายแกนทั้งหมดใน firmware

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
