# แถบสถานะ (Status Bar)

ตัวอย่างนี้สร้างแถบสถานะด้านล่างที่แสดงข้อมูลระบบ ได้แก่ ไอคอนสถานะ WiFi, ชื่อบอร์ด, เวลาทำงาน (uptime) และจำนวนเซ็นเซอร์ที่ใช้งานอยู่ แถบสถานะอัปเดตทุกวินาทีผ่าน LVGL timer

แถบเป็นคอนเทนเนอร์ความสูงคงที่ยึดไว้ด้านล่างของ parent ใช้ flex row layout เพื่อกระจายรายการอย่างสม่ำเสมอ ไอคอน WiFi เปลี่ยนระหว่างสัญลักษณ์เชื่อมต่อแล้ว (เขียว) และยังไม่เชื่อมต่อ (เทา) ตามสถานะ WiFi ที่จำลอง

ตัวนับ uptime จัดรูปแบบเวลาที่ผ่านไปเป็น HH:MM:SS โดยใช้ FreeRTOS tick count หารด้วย `configTICK_RATE_HZ` ชื่อบอร์ดถูกกำหนดตอน compile time ผ่าน preprocessor define แสดงให้เห็นวิธีที่ branding เฉพาะบอร์ดไหลผ่านไปยัง UI

จำนวนเซ็นเซอร์แสดงจำนวนเซ็นเซอร์ฮาร์ดแวร์ที่มีบนบอร์ดปัจจุบัน กำหนดตอน compile time โดยรวม BSP feature flag ต่างๆ ให้ผู้ใช้ทราบทันทีว่าเซ็นเซอร์ใดเข้าถึงได้ โดยไม่ต้องไปที่หน้าวินิจฉัย

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
