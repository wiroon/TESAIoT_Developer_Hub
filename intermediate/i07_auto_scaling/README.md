# กราฟปรับสเกลอัตโนมัติ (Auto-Scaling Chart)

ตัวอย่างนี้สาธิตการปรับสเกลแกน Y แบบไดนามิกที่ปรับตามค่าต่ำสุด/สูงสุดของหน้าต่างข้อมูลที่มองเห็นโดยอัตโนมัติ แทนที่จะใช้ช่วงคงที่ กราฟจะคำนวณขอบเขตใหม่ทุกครั้งที่อัปเดต ทำให้ข้อมูลเติมเต็มพื้นที่แนวตั้งที่มีอยู่เสมอ

หลังจากเพิ่มจุดข้อมูลใหม่แต่ละครั้ง timer callback จะสแกน array ค่า Y ภายในของกราฟโดยใช้ `lv_chart_get_y_array` เพื่อหาค่าต่ำสุดและสูงสุดปัจจุบัน จากนั้นเรียก `lv_chart_set_range` โดยเพิ่ม margin 10% ด้านบนและด้านล่างของค่าสุดขั้ว

เทคนิคนี้จำเป็นสำหรับเซ็นเซอร์ที่ช่วงเอาต์พุตเปลี่ยนแปลงมากตามสภาพแวดล้อม ช่วงคงที่ที่เหมาะกับการเอียงเบาๆ จะตัดสัญญาณเมื่อเขย่าแรง ขณะที่ช่วงกว้างพอสำหรับการเขย่าจะทำให้การเคลื่อนไหวเบาๆ มองไม่เห็น

label ช่วงมุมขวาบนแสดงขอบเขตแกน Y ปัจจุบัน ให้นักพัฒนาสังเกตการปรับสเกลอัตโนมัติแบบเรียลไทม์ ตัวอย่างใช้ BMI270 accelerometer แกน X เป็นแหล่งข้อมูล series เดียว

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
