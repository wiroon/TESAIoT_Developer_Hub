# ชุดสีธีม (Theme Colors)

ตัวอย่างนี้แสดงค่าคงที่ `UI_COLOR` ทั้งหมดที่กำหนดไว้ใน `tesaiot_ui_theme.h` ในรูปแบบ grid อ้างอิงสี แต่ละ swatch แสดงค่า hex และชื่อความหมาย (semantic name) ทำหน้าที่เป็น style guide แบบสดสำหรับการพัฒนา BENTO UI

จานสีประกอบด้วยสีเน้นเฉพาะเซ็นเซอร์ (BMI270 เขียว, DPS368 ส้ม, SHT40 น้ำเงิน, BMM350 ม่วงแดง, Radar แดง, Joystick เหลืองอำพัน) สีพื้นหลัง (การ์ดน้ำเงินเข้ม, แผงสีเข้ม) และสีสถานะ (สำเร็จ เขียว, เตือน เหลือง, ข้อผิดพลาด แดง)

แต่ละ swatch เป็นคอนเทนเนอร์เล็กๆ ที่ใช้สีนั้นเป็นพื้นหลัง พร้อม label แสดงค่า hex และ label แสดงชื่อ swatch ทั้งหมดจัดเรียงในเลย์เอาต์แบบ grid ให้ภาพรวมอ้างอิงที่นักพัฒนาสามารถใช้ขณะสร้างหน้า dashboard

ตัวอย่างนี้ไม่ต้องใช้ฮาร์ดแวร์เซ็นเซอร์และทำงานได้บนทุกบอร์ด เป็นเครื่องมืออ้างอิงภาพล้วน แต่สาธิตเทคนิค LVGL ที่สำคัญ ได้แก่ grid layout, การจัดการสีแบบ programmatic และรูปแบบการสร้าง UI จาก data array แทนการสร้าง widget แบบ hardcode
