# ข้อมูล Gyroscope แบบเรียลไทม์ (Real-Time Gyroscope)

ตัวอย่างนี้แสดงผลเอาต์พุต BMI270 gyroscope แบบเรียลไทม์ โดยพล็อตความเร็วเชิงมุมรอบแกน X, Y และ Z รูปแบบเหมือนกับ I04 (accelerometer) แต่ใช้ข้อมูล gyroscope ในหน่วยองศาต่อวินาทีแทน milli-g

ช่วงแกน Y ของกราฟตั้งไว้ที่ -500 ถึง +500 dps (องศาต่อวินาที) ตรงกับค่า config ของ BMI270 gyro ทั่วไป มี 3 series แยกสี (แดง=X, เขียว=Y, น้ำเงิน=Z) ติดตามการเคลื่อนที่แบบหมุนตลอด 100 จุดตัวอย่าง

timer ที่ทำงานทุก 50ms อ่าน register gyroscope และส่งค่าความเร็วเชิงมุมเข้ากราฟ label แสดงค่าตัวเลขใต้กราฟแสดงค่า dps ปัจจุบัน การแสดงผล 2 ชั้นนี้ (กราฟ + label) ให้นักพัฒนาเห็นทั้งแนวโน้มและค่าปัจจุบันที่แม่นยำ

ข้อมูล gyroscope มีสัญญาณรบกวนมากกว่า accelerometer เมื่อบอร์ดอยู่นิ่ง นักพัฒนาจะสังเกตเห็นเส้นกราฟกระตุกรอบศูนย์เมื่อบอร์ดหยุดนิ่ง นี่เป็นพฤติกรรมปกติของเซ็นเซอร์ และแสดงให้เห็นว่าทำไมการกรองข้อมูล (ครอบคลุมในตัวอย่างขั้นสูง) จึงสำคัญสำหรับแอปพลิเคชันจริง
