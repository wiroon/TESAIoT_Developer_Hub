# โรงงานเกจ Arc (Arc Gauge Factory)

ตัวอย่างนี้จัดเตรียมฟังก์ชัน `tesaiot_arc_gauge_create()` ที่ใช้ซ้ำได้ สำหรับสร้าง widget เกจ arc แบบครบชุด ประกอบด้วย label ชื่อเรื่อง, ตัวบ่งชี้ arc และการแสดงค่าตรงกลาง ฟังก์ชันคืนค่า struct ที่มี handle ของทั้ง arc และ label ค่า เพื่อให้อัปเดตได้ง่าย

ฟังก์ชัน factory รับพารามิเตอร์สำหรับชื่อเรื่อง, สีเน้น, ช่วงค่า (min/max), ขนาด arc และ widget แม่ arc ถูกกำหนดค่าด้วยมุมกวาด 270 องศา (135 ถึง 45 องศา) ตรงกับรูปลักษณ์เกจมาตรฐานที่ใช้ทั่วทั้ง BENTO dashboard

label ค่ากลางถูกวางตำแหน่งภายใน arc ซ้อนทับจุดศูนย์กลาง label ชื่อเรื่องถูกวางไว้เหนือ arc ตัวบ่งชี้ arc ใช้สีเน้นที่กำหนด และ track พื้นหลังใช้สีเข้มเพื่อความคมชัด

ตัวอย่างสร้างเกจ 4 ตัวเรียงกัน ด้วยสีและช่วงค่าที่แตกต่างกัน ได้แก่ อุณหภูมิ (0-50), ความดัน (900-1100), ความชื้น (0-100) และแบตเตอรี่ (0-100) แต่ละตัวตั้งค่าเริ่มต้นไว้เพื่อสาธิต ในการใช้งานจริง timer callback จะอัปเดตค่าจากการอ่านเซ็นเซอร์

รองรับทั้ง TESAIoT Dev Kit, PSoC Edge AI Kit และ PSoC Edge Eva Kit
