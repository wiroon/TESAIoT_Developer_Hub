# กราฟเส้นเดี่ยว (Single Line Chart)

ตัวอย่างนี้สร้างกราฟเส้นแบบเรียลไทม์ที่พล็อตจุดข้อมูลแบบสุ่มอย่างต่อเนื่องโดยใช้ LVGL timer กราฟแสดงข้อมูล 100 จุดในแต่ละครั้ง โดยค่าใหม่จะถูกเพิ่มทุกรอบ timer ทำให้หน้าต่างที่มองเห็นเลื่อนไปทางซ้ายเมื่อข้อมูลสะสมมากขึ้น

กราฟถูกกำหนดค่าด้วย `lv_chart_set_type` สำหรับโหมดเส้น และ `lv_chart_set_point_count` เพื่อเก็บ 100 ตัวอย่าง มีการเพิ่ม series เดียวด้วย `lv_chart_add_series` โดยใช้แกน Y หลัก ช่วงแกน Y ถูกตั้งไว้ที่ 0-100 เพื่อให้ตรงกับช่วงค่าสุ่ม

LVGL timer callback ที่ทำงานทุก 200ms จะสร้างค่าสุ่มใหม่แต่ละรอบและส่งไปยังกราฟผ่าน `lv_chart_set_next_value` ฟังก์ชันนี้จัดการ circular buffer ภายในโดยอัตโนมัติ ทำให้จุดเก่าเลื่อนหายไปทางซ้ายเมื่อจุดใหม่ปรากฏทางขวา

มี label ชื่อเรื่องและค่าสดแสดงอยู่เหนือกราฟ label ค่าจะอัปเดตทุกรอบ timer เพื่อแสดงจุดข้อมูลล่าสุดเป็นตัวเลข ให้ผู้ใช้เห็นผลตอบรับทั้งแบบกราฟิกและแบบตัวอักษรพร้อมกัน
